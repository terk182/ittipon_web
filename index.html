Features

จะได้อะไรกลับไปจากการเรียน

สิ่งที่จะได้ 1

ปูพื้นฐานการเขียนโปรแกรมเพื่อเข้าใจหลักการทำงานและการนำไปใช้แก้ปัญหาในแบบต่างๆ

สิ่งที่จะได้ 2

ได้รู้วิธีการเขียนโปรแกรมและสร้างโปรแกรมเพื่อเป็นจุดเริ่มเพื่อเป็นแนวคิดเพื่อพัฒนาตนเองต่อได้

สิ่งที่จะได้ 3

ใช้เครื่องมือสำหรับเขียนโปรแกรมที่นิยมในปัจจุบัน เช่น visual studio code และการใช้ git

สิ่งที่จะได้ 4

แนะแนวทางการต่อยอดเพื่อเป็นโปรแกรมมืออาชีพ เติมเต็มเทคนิคเพื่อเข้าสู่การทำงานจริง

สิ่งที่จะได้ 5

ประยุกต์ใช้งานโปรแกรมคอมพิวเตอร์เพื่อแก้ปัญหาหรือความต้องการในชีวิตประจำวันหรือเพื่อธุรกิจ

สิ่งที่จะได้ 6

ใช้งานระบบฐานข้อมูลและติดตั้งระบบเพื่อเผยแพร่โปรแกรมต่อสาธาระณะได้

Faq คำถาม

Questions

ช่วงอายุเท่าไร ต้องมีพื้นฐานขนาดใหน?

ใช้คำว่าผู้สนใจจะดีกว่าหาก พี่ๆน้องๆลูกๆหลานๆสนใจ ก็สามารถเรียนได้ แต่ต้องใช้คอมพิวเตอร์ได้ และสนใจใน การโปรแกรมคอมพิวเตอร์

ต้องเตรียมตัวยังไงบ้าง?

คอมหนึ่งเครื่องและเวลาว่าง เรามาก็ลุยกันได้.

เรียนแถวใหน?

ไม่มีสถาฐานที่ที่แน่นอนครับ สอนตามร้านกาแฟ ร้านกาแฟในเมืองน่านเยอะนะ ผู้สนใจสามารถเลือกเวลาและสถานที่ได้ จากเว็ปนี้ได้เลย

เรียนนานมั้ย ยากมั้ย?

เรียนไม่นาน เพราะผู้สอนก็ทำงานในสายงานอยู่ และไม่ยากขนาดนั้น ทุกคนเรียนรู้ได้ แล้วจะชอบมันนะ การโปรแกรมเนี้ย
